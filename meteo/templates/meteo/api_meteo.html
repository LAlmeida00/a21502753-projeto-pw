API Meteorologia

API Meteorologia PW2024

Esta página oferece acesso a uma API personalizada, fornecendo uma variedade de informações úteis para desenvolvedores e utilizadores que desejam utilizar em suas próprias aplicações.

Sobre a API:

Esta API foi feita com intuito académico, oferecendo uma série de endpoints para adquirir conhecimento na criação de uma API personalizada.

Cada endpoint é projetado para fornecer informações detalhadas sobre diferentes aspectos da API.

Endpoints

1. Lista de Cidades

Metodo: GET

URL:/api/cities/

Descrição: Retorna uma lista de todas as cidades.

2. Previsao Meteorologica de Hoje de uma cidade

Metodo: GET

URL:/api/weather/today/{string}/

Descrição: Retorna a previsao meteorologica de hoje, de uma cidade (Exemplo: Lisboa)

Parâmetros:

  • city (string): cidade desejada
  • temp_min (integer): temperatura mínima
  • temp_max (integer): temperatura maxima
  • data (date): Data da previsao ou registro
  • weather (string): descricao do estado do tempo
  • icon_url (string): URL do icone do estado do tempo

3. Previsao Meteorologica de 5 dias de uma cidade

Metodo: GET

URL:/api/weather/5days/{string}/

Descrição: Retorna a previsao meteorologica dos proximos 5 dias de uma cidade (Exemplo: Lisboa)

  • city (string): cidade desejada
  • temp_min (integer): temperatura mínima
  • temp_max (integer): temperatura maxima
  • data (date): Data da previsao ou registro
  • weather (string): descricao do estado do tempo
  • icon_url (string): URL do icone do estado do tempo
